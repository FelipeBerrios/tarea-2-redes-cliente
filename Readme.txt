Existen 2 versiones, una que se trabajó en Windows y es el proyecto en Netbeans directamente exportado a GitHub
Por otro lado está el proyecto Make que se puede ejecutar en cualquier SO basado en Unix, por alguna razón funciona en Chrome y Mozilla
En Windows pero solo en Mozilla en Fedora donde fue probado.
 
Para compilar en Windows hacer pull y abrir el proyecto de la carpeta ServidorJava, y luego compilar normalmente.
Para Linux hacer pull y ubicarse en el directorio TareaMake/servidorjava y ejecutar el comando make, luego de que compile, ejecutar 
Con  el comando "java ServidorJava".

En este caso se utilizó el puerto 8000, entonces para abrir utilizar http://localhost:8000 en la dirección del navegador.
